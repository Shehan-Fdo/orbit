# Semantic HTML

Semantic HTML means using the right element for the right job.

## Why It Matters

- Improves **accessibility** (screen readers)
- Better **SEO**
- Easier to read and maintain

## Common Semantic Elements

| Element | Purpose |
|---------|---------|
| `<header>` | Top of page or section |
| `<nav>` | Navigation links |
| `<main>` | Primary content |
| `<article>` | Self-contained content |
| `<section>` | Grouped content |
| `<aside>` | Sidebar content |
| `<footer>` | Bottom of page or section |

## Example

```html
<body>
  <header>
    <nav>...</nav>
  </header>
  <main>
    <article>
      <h1>Post Title</h1>
      <p>Content here...</p>
    </article>
    <aside>Related links</aside>
  </main>
  <footer>© 2026</footer>
</body>
```
