# HTML Overview

HTML (HyperText Markup Language) is the backbone of every webpage. It defines the structure and meaning of web content.

## What HTML Does

- Defines headings, paragraphs, links, images
- Gives semantic meaning to content
- Works alongside CSS and JavaScript

## Basic Document Structure

```html
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8" />
    <title>My Page</title>
  </head>
  <body>
    <h1>Hello, World!</h1>
  </body>
</html>
```

> HTML is not a programming language — it's a markup language.
